# OptimalWeightCalculator – вычисление оптимального веса

WPF-приложение для оценки веса человека на основе роста, веса и пола с учётом допустимого отклонения ±3 кг от нормы.

## Функциональность
- Ввод роста (130–220 см) и веса (40–170 кг).
- Выбор пола (мужчина/женщина).
- Расчёт нормального веса:
  - Мужчины: (рост - 100) × 1.13
  - Женщины: (рост - 100) × 0.9
- Вывод характеристики: «Ниже нормы», «Норма», «Выше нормы» с учётом ±3 кг.
- Проверка входных данных на корректность и диапазоны.
